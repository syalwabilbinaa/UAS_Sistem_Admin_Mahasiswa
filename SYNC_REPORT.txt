🐛 TROUBLESHOOTING

Problem: Application shows 404
Solution: Check that all files are in public/ folder
          Verify vercel.json routing configuration

Problem: Google Sheets not syncing
Solution: Verify URL in public/app.js line 2
          Check browser console for errors
          Test Google Apps Script endpoint directly

Problem: Port 3000 already in use
Solution: npm start -- --port 3001
          Or kill process using: lsof -ti:3000 | xargs kill -9
